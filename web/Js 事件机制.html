# 前言

JS的事件机制有宏任务和微任务。

宏任务是setTimeout、requestAnimationFrame、用户输入事件（I/O）等，它是由浏览器的队列完成的，在浏览器的主进程中进行，页面不会卡死。

微任务，包括Promise、MutationObserver等，是在浏览器的主线程中执行的，

主线程在当前任务快要执行结束之前，检查微任务队列中是否存在微任务，如果有，那么那么那么当前任务会依次取出微任务队列中的微任务，并一一执行！
